# 🌍 Diplomacia de Dados: Laboratório Geopolítico

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://portfolio-geopolitica.streamlit.app)

## 📌 Sobre o Projeto
Este projeto nasceu da intersecção entre a **Ciência de Dados** e as **Relações Internacionais**. Mais do que um simples dashboard técnico, trata-se de um laboratório analítico projetado para compreender como a formação de blocos econômicos (como G7, BRICS e Mercosul) e eventos históricos globais moldam o desenvolvimento e a distribuição de riqueza entre as nações.

**👉 [Acesse a aplicação ao vivo aqui](https://portfolio-geopolitica.streamlit.app)**

## ⚙️ Funcionalidades
* **Filtros Interativos:** Seleção dinâmica de blocos econômicos para análise comparativa limpa e focada.
* **Análise Histórica Linear:** Gráficos temporais sinalizando marcos geopolíticos cruciais (Ex: Crise do Petróleo de 1973, Criação do Mercosul em 1991, Expansão da China na OMC em 2001).
* **Mapeamento Geoespacial:** Visão em mapa-múndi (`choropleth`) atualizada com os dados de PIB per capita mais recentes do dataset.
* **Data Storytelling:** Painéis expansíveis contendo análises de conjuntura internacional escritas de forma autoral, unindo teoria política e dados quantitativos.

## 🛠️ Tecnologias Utilizadas
* **Python:** Lógica de programação e manipulação de dados.
* **Pandas:** Estruturação, limpeza e agrupamento de dados (`DataFrames`).
* **Plotly (Express):** Visualização de dados interativa e de alta fidelidade (gráficos de linha e mapas coropléticos).
* **Streamlit:** Construção rápida do front-end (dashboard web) e deploy na nuvem.

## 🚀 Como rodar o projeto localmente
Caso queira clonar este repositório e rodar no seu computador:

1. Clone o repositório:
```bash
git clone [https://github.com/seu-usuario/portfolio-geopolitica.git](https://github.com/seu-usuario/portfolio-geopolitica.git)
```
2. Instale as dependências:
```bash
pip install -r requirements.txt
```
3. Execute o aplicativo:
```bash
streamlit run app.py
```
